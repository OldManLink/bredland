# BRD-008 — Monthly telemetry archive consolidation

## Status

Complete

## Goal

Consolidate the previous month's daily telemetry archives into a single monthly archive.

Daily telemetry rotation already produces one compressed JSONL archive (`.jsonl.gz`) per host per day. While this keeps the live logs small, a month's worth of daily archives quickly becomes cumbersome to browse and manage.

This slice introduces a second archival stage that automatically consolidates the previous month's daily archives into a single compressed monthly archive.

---

## Design

### Monthly consolidation

At 03:30 on the first day of each month, the previous month's daily archives for each monitored host shall be consolidated into a single archive.

The daily `.jsonl.gz` files shall first be decompressed and concatenated into a single JSONL stream before being compressed into a new monthly archive.

This avoids creating nested gzip archives while preserving the original JSONL format.

The resulting archive shall be named:

```
YYYY-MM.jsonl.gz
```

Once the monthly archive has been successfully created, the original daily archives for that month may be removed.

---

## Operational Note

The monthly consolidation script is intended to run from cron at 03:30 on the first day of each month.

When run normally on the first day of a month, it consolidates the previous month.

For manual rehearsal or emergency runs on any other date, set:

```bash
NOC_CONSOLIDATION_DATE=YYYY-MM-01
```

For example, to consolidate June 2026 manually:

```bash
NOC_CONSOLIDATION_DATE=2026-07-01 ./consolidate-monthly-logs.sh
```

---

## Acceptance Criteria

* Consolidation runs automatically at 03:30 on the first day of each month. ✅
* All daily archives for the previous month are combined into a single monthly archive. ✅
* The monthly archive is a normal gzip-compressed JSONL file (not nested gzip archives). ✅
* The monthly archive is named `YYYY-MM.jsonl.gz`. ✅
* Daily archives are removed only after successful consolidation. ✅
* Normal heartbeat processing is unaffected throughout the operation. ✅
